# HierarchyPanel

## Source

- `srcs/plugins/user_interface/panels/HierarchyPanel.hpp`

## Role

Displays camera/lights/primitives tree-like list and handles selection.

## Notable Features

- Multi-selection support.
- Camera selection handling.
- Selection sync with viewport picking.
- Hide request callback (`setOnItemHideRequest`).
- Internal scroll and scrollbar handling fields.
