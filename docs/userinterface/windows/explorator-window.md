# ExploratorWindow

## Source

- `srcs/plugins/user_interface/windows/ExploratorWindow.hpp`

## Role

File/folder explorer window used for scene open/save path selection.

## Modes

- `OPEN`
- `SAVE`

## Features

- Directory listing with parent navigation (`..`).
- Hidden files filtering.
- Filename field in save mode.
- Confirmation button (`Open` / `Save`).
