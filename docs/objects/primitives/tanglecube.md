# Tanglecube Primitive

## Description

The tanglecube is an implicit algebraic surface. It is a smooth high-genus shape generated by a quartic polynomial field and an isosurface threshold.

## Mathematical Definition

In local coordinates `p = (x, y, z)`, the scalar field is:

```text
f(p) = x^4 - 5x^2 + y^4 - 5y^2 + z^4 - 5z^2 + threshold
```

Surface is the zero level set:

```text
f(p) = 0
```

## Ray-Intersection Calculation

Ray in local object space:

```text
p(t) = o + t d
```

Substitution gives a quartic-in-components scalar function:

```text
g(t) = f(o + td)
```

Rather than solving full quartic analytically, implementation performs:

1. Coarse bounding-sphere entry/exit interval (`tEntry`, `tExit`).
2. Marching with adaptive step size to detect sign change `g(t_prev) * g(t_cur) < 0`.
3. Root refinement:
   - Newton updates when derivative is stable.
   - Bisection fallback for robustness.

Intersection is valid when refined `t` is in `[t_min, t_max]`.

## Normal Computation

Normal uses gradient of implicit field:

```text
grad f = (
    4x^3 - 10x,
    4y^3 - 10y,
    4z^3 - 10z
)
```

Then:

```text
N = normalize(grad f(localHit))
```

## Edge Cases

- No sign change on sampled interval: no hit.
- Near-flat derivative: Newton may fail; bisection fallback is needed.
- Too large step can skip thin features.
- Threshold values can make shape disappear or split.

## Pseudocode

```pseudocode
function intersectTanglecube(ray, center, size, threshold, tMin, tMax):
    o = (ray.origin - center) / size
    d = ray.direction / size

    [tEntry, tExit] = intersectBoundingSphere(o, d)
    clamp to [tMin, tMax]
    if empty interval: return no_hit

    tPrev = tEntry
    valPrev = f(o + tPrev*d)

    for tCur from tEntry to tExit with adaptive steps:
        valCur = f(o + tCur*d)
        if valPrev * valCur < 0:
            tRoot = refineRootByNewtonThenBisection(tPrev, tCur)
            localH = o + tRoot*d
            N = normalize(gradF(localH))
            H = ray.at(tRoot)
            return hit_record(tRoot, H, N)

        tPrev = tCur
        valPrev = valCur

    return no_hit
```
