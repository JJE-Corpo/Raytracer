Accueil / Référence

Référence des classes

Interfaces et classes principales du moteur. Les interfaces (préfixe I…) définissent les contrats ; les implémentations concrètes vivent dans srcs/core et srcs/plugins.

Interfaces principales

IPlugin srcs/common/IPlugin.hpp

Contrat de base de tout plugin dynamique.

virtual PluginType getType() const
Retourne le type du plugin : RENDERER ou USER_INTERFACE.

ISceneRenderer srcs/common/ISceneRenderer.hpp

Contrat d’un renderer (hérite de IPlugin).

void renderScene(const IScene &scene)
Rend la scène fournie.
void setPixel(int x, int y, Color color)
Écrit un pixel dans le tampon.
void stopRendering()
Interrompt le rendu en cours.
bool isRendering() const
Indique si un rendu est en cours.
int getCurrentSample() const
Numéro de l’échantillon courant (rendu progressif).
Render getRender() const
Retourne le rendu (dimensions + pixels).
std::string getRendererName() const
Nom du renderer (« Default », « Viewport »).

IScene srcs/common/scene/IScene.hpp

Conteneur des données de scène et point d’intersection global.

void addPrimitive(IPrimitive *) / void addLight(ILight *)
Ajoute une primitive / une lumière.
void addDefaultPrimitive(std::string type) / addDefaultLight(std::string type)
Crée un objet par défaut à partir d’un type.
ISceneObject *addGroup()
Crée un groupe (hiérarchie).
void reparent(child, newParent, index)
Change le parent d’un objet.
void removeObject(ISceneObject *)
Supprime un objet.
IPrimitive *convertToMesh(IPrimitive *)
Convertit une primitive en maillage éditable.
Material *getMaterial(name) / createMaterial(...)
Gestion des matériaux.
float get/set AmbientCoefficient / DiffuseCoefficient
Coefficients d’éclairage global.
ICamera &getCamera() const
Accès à la caméra.
const std::vector<IPrimitive*> &getPrimitives() const
Liste des primitives (idem lumières, matériaux, racines).
void buildBvh()
(Re)construit la structure d’accélération BVH.
bool intersect(ray, tMin, tMax, hit) const
Teste l’intersection la plus proche.

ICamera srcs/common/scene/ICamera.hpp

Caméra perspective : paramètres et génération de rayons.

Vector2i getResolution() / Vector3f getPosition() / getRotation() / double getFov()
Accès aux paramètres.
Vector3f getForward() / getRight()
Vecteurs de base de la caméra.
int getSamplesPerPixel() / double getPixelSamplesScale()
Échantillonnage anti-aliasing.
Ray generateRay(int i, int j) const
Génère le rayon primaire du pixel (i, j).

IPrimitive srcs/common/scene/IPrimitive.hpp

Contrat de toute surface intersectable (hérite de ISceneObject).

bool intersect(const Ray&, float tMin, float tMax, Intersection&) const
Teste l’intersection dans [tMin, tMax] et remplit le hit.
AABB bounding_box() const
Boîte englobante (pour le BVH).
bool isFinite() const
Faux pour un objet infini (ex. un plan).
std::map<std::string, std::pair<std::string, PropertyType>> getProperties() const
Propriétés éditables (nom → valeur + type).
void setPropertyFloat(const std::string &key, float value)
Modifie une propriété scalaire.
const Material *getMaterial() / void setMaterial(...)
Matériau associé.

Types de propriété (PropertyType) : COLOR, VECTOR3F, VECTOR3I, VECTOR2F, VECTOR2I, FLOAT, INT, STRING.

IEditablePrimitive srcs/common/scene/IEditablePrimitive.hpp

Capacité optionnelle : une primitive éditable au sommet (Triangle, Mesh). Découverte à l’exécution via dynamic_cast<IEditablePrimitive*>, ce qui laisse IPrimitive inchangé.

ILight srcs/common/scene/ILight.hpp

LightKind getKind() const
POINT, DIRECTIONAL ou UNKNOWN.
float getIntensity() / void setIntensity(float)
Intensité.
ColorF getColorF() / void setColorF(...)
Couleur.

IPluginLoader srcs/common/IPluginLoader.hpp

Contrat du chargeur de plugins : getPlugins(), getPlugins(type), unloadAll().
